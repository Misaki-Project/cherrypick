# Misaki-Project/cherrypickの独自機能

* レベルロール機能
	* ユーザーのロールに値を持つことができます
	* 柔軟なレベルアップの条件が指定可能
	* レベル毎に異なるポリシーの条件を指定することが可能
* ロール
	* ロールの説明にMFMや改行を利用できるように
	* プロフィールのバッジ上から特定のロールを非表示にする機能
* ポリシー
	* 投稿のパブリック可能な条件がより詳細に
		* 通常・自己RN・ローカルRN・リモートRN・引用・ファイル付のパブリック投稿の可否
	* フォロー上限数の指定をできるように
		* 基礎上限・フォロワーの倍率
	* ユーザーがアカウント削除してもデータ上残すオプションの追加
	* アカウント削除・アカウント整理ができるかのオプションの追加
	* リアクションの全体的な制限が可能に
		* 全て、センシティブを除く、Unicode絵文字のみ、ハートのみ、拒否
* 投稿毎の連合制御
	* ユーザーが送信したくないサーバーに配送を制限できるように。
* ロックダウンモード
	* ピン止めのみロックダウンモードの対象外として誰でも見れるように
* ユーザーページ
	* その人にフォローされていて且つ自分がフォローしているユーザーを表示する機能
* drop&fusion
	* ランキング集計が1時間から1年まで遡る機能
* チャット
	* ユーザー同士のチャットの上部にユーザーのプロフィールを表示する機能
	* ブロックした、されたユーザーのチャットやリアクションの送信ができないように
* モデレーション
	* IPに紐づけられたユーザーを検索する機能 (仮実装)
* クライアント
	* 非公開の投稿を閲覧した場合サーバー設定で指定した文字列に置き換える機能。
* 連合
	* ユーザーがサーバー単位で配送するオプションを追加
